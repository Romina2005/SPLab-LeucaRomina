# bookmodel-lab5
Spring Boot project for Design Patterns Lab 5-6 (Spring Data JPA + H2).

## How to run
- Import as a Gradle project in IntelliJ (or run `./gradlew bootRun` if you add the Gradle wrapper).
- Run the main class `bookmodel.BookApplication`.
- H2 console: http://localhost:8080/h2-console
  - JDBC URL: jdbc:h2:file:~/h2db/booksdb
  - User: sa, password: (empty)
